# 安卓app开发模板  
+ 简介  
　　封装一个通用型的app模板：  
　　　　1. 以后续app开发时可以快速进入项目细节开发  
　　　　2. 规范项目组织结构  
　　　　3. 封装常用的技术点，以期望最大化达到重用、方便  
　　　　4. 持续化学习  
+ 内容  
　　模板包含内容：  
　　　　1. 封装网络操作框架，简化网络请求。采用retrofit+okhttp+rxandroid架构，封装常见操作，如：日志输出、HTTPS设置、参数加密、常见错误处理  
　　　　2. 进程/进程、进程/线程通讯：EventBus  
　　　　3. 注解框架：butterkinfe  
　　　　4. 数据库Orm框架  
　　　　5. 抽象类：BaseActivity
***
